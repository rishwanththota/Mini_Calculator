# 🧮 Mini Calculator (HTML, CSS, JavaScript)

A simple and elegant **Mini Calculator** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean, responsive, and modern dark-themed interface.

---
## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3 (Grid + Flexbox)** – Styling & layout
* **JavaScript (ES6)** – Logic & functionality

---

## 📂 Project Structure

```
mini-calculator/
│── index.html      # Main file (HTML + CSS + JS)
│── README.md       # Documentation
```

---
## 💡 How It Works

* The UI is designed using **CSS Grid layout**.
* User interactions are handled via JavaScript functions:

  * `press(value)` → Appends input to display
  * `clearDisplay()` → Clears the screen
  * `calculate()` → Evaluates the expression
* Error handling is managed using `try...catch`.
